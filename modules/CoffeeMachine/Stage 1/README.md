# Coffee Machine

Simple interact with user for making 1 type of coffee using existing components

##### Interaction example:

In case of exit:
```html
Hello, how can I help you?
1. Make coffee
0. Exit
0
Have a good day!
```

In case of making coffee:

```html
Hello, how can I help you?
1. Make coffee
2. Service mode
0. Exit
1
Good choice, enjoy your coffee!
```
